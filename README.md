# Telegram Autoposter Bot 🤖

Многофункциональный Telegram-бот для автоматической публикации контента в каналы. Позволяет пользователям настроить очередь публикаций с настраиваемыми задержками, управлять черновиками и многое другое.

## 🚀 Основные возможности

- **Очередь публикаций**: Автоматическая публикация постов со случайной задержкой (настраивается).
- **Мультиязычность**: Поддержка русского, английского и узбекского языков.
- **Веб-панель управления**: Удобный интерфейс для управления очередью постов через браузер.
- **Интерактивные черновики**: Управление постами через кнопки "Опубликовать сейчас" и "Удалить".
- **Реферальная система**: Приглашение друзей и отслеживание статистики.
- **Админ-панель**: Рассылка сообщений и просмотр статистики пользователей.
- **Донаты**: Поддержка бота через Telegram Stars.
- **Локальное хранилище медиа**: Все файлы хранятся на сервере, без необходимости в дополнительных каналах.

## 🐳 Docker

Быстрый запуск с помощью Docker Compose:

```bash
# Скопируйте файл окружения
cp .env.docker .env

# Отредактируйте .env и укажите ваши данные

# Запустите бота и базу данных
docker-compose up -d

# Просмотр логов
docker-compose logs -f bot
```

Веб-панель будет доступна по адресу `http://localhost:8080/panel`

## 🛠️ Технологический стек

- **Язык**: Python 3.10+
- **Основная библиотека**: Aiogram 3
- **База данных**: PostgreSQL
- **Асинхронный драйвер БД**: asyncpg

## ⚙️ Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШ_НИК/autoposter-bot.git
    cd autoposter-bot
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    
3.  **Настройте PostgreSQL:**
    Убедитесь, что у вас запущена база данных PostgreSQL, к которой бот сможет подключиться.

4.  **Настройте переменные окружения:**
    Скопируйте файл `.env.example` в `.env` и заполните необходимыми значениями:
    ```bash
    cp .env.example .env
    ```
    
    Отредактируйте `.env` файл:
    - `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather
    - `DATABASE_URL` - URL подключения к PostgreSQL
    - `ADMIN_IDS` - Telegram ID администраторов (через запятую)
    - `PANEL_HOST`, `PANEL_PORT`, `PANEL_BASE_URL` - настройки веб-панели
    - `AUTO_PUBLISH_DELAY_MIN`, `AUTO_PUBLISH_DELAY_MAX` - задержки автопубликации (в секундах)

5.  **Запустите бота:**
    ```bash
    python main.py
    ```

## 🔧 Конфигурация

Все настройки задаются через переменные окружения или файл `.env`:

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота | - |
| `DATABASE_URL` | URL PostgreSQL базы данных | - |
| `ADMIN_IDS` | ID администраторов (через запятую) | - |
| `PANEL_HOST` | Хост веб-панели | `0.0.0.0` |
| `PANEL_PORT` | Порт веб-панели | `8080` |
| `PANEL_BASE_URL` | Базовый URL веб-панели | `http://127.0.0.1:8080` |
| `AUTO_PUBLISH_DELAY_MIN` | Минимальная задержка (сек) | `1800` (30 мин) |
| `AUTO_PUBLISH_DELAY_MAX` | Максимальная задержка (сек) | `3600` (60 мин) |
